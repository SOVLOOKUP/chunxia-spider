# oneanother爬虫平台

简单的前后端交互爬虫，前端可显示图片，按钮一键爬取，部署方便

## 安装

1. 安装docker
2. `git clone --recurse-submodules https://github.com/SOVLOOKUP/chunxia-spider` 
3. `cd chunxia-spider && sudo sh ./install.sh`
